# Overview
Danish translation for the open source control panel Pterodactyl.
Supported version: v1.11.2

# Warning
These files are a full copy of the resources folder in Pterodactyl, and therefore also contain a lot more than just the language.
I will update it when I have time.

Therefore, errors may occur if you use these files right now and here on a higher version than I have done it on.

# How to install
Overwrite all the files in /var/www/pterodactyl and follow the guide here: https://pterodactyl.io/community/customization/panel.html
